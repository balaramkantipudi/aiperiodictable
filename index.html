The Periodic Table of Agentic & GenAI

A periodic-table-style map of the most useful concepts across LLM foundations, generation, RAG, agents, evaluation, safety, and deployment. Each tile is a keyword cluster, navigable by topic and category.
Foundation Generation Retrieval Agentic Ops/Evals Safety Infra
01
LLM
Large Language Model
Neural model trained on massive text corpora.
02
Tr
Transformer
Architecture behind modern language models, built on attention.
03
At
Attention
Mechanism that weighs token relationships across a sequence.
04
Tk
Tokenization
Breaking text into tokens the model can process.
05
Inf
Inference
Running a model to predict the next token and generate output.
06
Pr
Prompt Engineering
Crafting instructions, format, and examples for better results.
07
FS
Few-Shot
Giving examples in the prompt to improve task performance.
08
FT
Fine-Tuning
Adapting a pretrained model on narrower task data.
09
Tmp
Temperature
Controls randomness during generation.
10
MM
Multimodal
Models that process or generate text, image, audio, or video.
11
RAG
Retrieval-Augmented Generation
Grounds model output with external knowledge sources.
12
Emb
Embeddings
Dense vector representations that capture semantic meaning.
13
VDB
Vector Database
Stores and queries high-dimensional vectors efficiently.
14
Ch
Chunking
Splitting content into smaller retrieval units.
15
Hy
Hybrid Search
Combines semantic and keyword search strategies.
16
RR
Reranking
Reorders retrieved results for stronger relevance.
17
MMR
Max Marginal Relevance
Balances relevance with diversity in results.
18
Ag
Agent
LLM-based system with goals, tools, memory, and actions.
19
AR
Agentic RAG
RAG enhanced with tool use, planning, and validation.
20
ReA
ReAct
Reason + Act loop: thought, action, observation, repeat.
21
Pln
Planning
Breaking a task into steps before acting.
22
TUl
Tool Use
Calling APIs, search, calculators, or software functions.
23
Fn
Function Calling
Structured way for models to invoke external tools.
24
Mem
Memory
Short-term and long-term state across interactions.
25
Rt
Router
Selects the best tool, agent, or knowledge source.
26
MAS
Multi-Agent System
Multiple specialized agents coordinate on a task.
27
Obs
Observation
Feedback an agent receives after acting.
28
Ref
Reflection
Self-critique or re-evaluation to improve outcomes.
29
Eval
Evaluation
Measures whether retrieval or generation quality meets needs.
30
RGE
RAG Evaluation
Assesses retrieval quality and grounded answer relevance.
31
Exp
Explainability
Showing sources or reasoning traces behind outputs.
32
Hal
Hallucination
Fluent output that is wrong, unsupported, or fabricated.
33
Grd
Guardrails
Rules and constraints that keep outputs and actions safe.
34
HITL
Human-in-the-Loop
Human review or intervention inside automated workflows.
35
OSS
Open-Source LLM
Model with publicly available weights for local operation.
36
Ctx
Context Window
Maximum information a model can consider in one pass.
37
Lat
Latency
Delay between input and usable model response.
38
A2A
Agent-to-Agent
Protocols or coordination patterns for agents to communicate.
39
Wrk
Workflow
Structured sequence of model, tool, and human steps.
40
Orc
Orchestration
Coordinating agents, tools, prompts, and control flows.
How to use it: read left-to-right as a learning map: foundations → generation → retrieval → agents → evaluation/safety → infrastructure.
Best for: LinkedIn posts, interview prep, workshop slides, personal notes, and a portfolio visual about the GenAI landscape.
